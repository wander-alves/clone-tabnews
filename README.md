# Projeto Clone Tabnews
Criei esse repositório com base nas aulas do [Curso.Dev](https://curso.dev) do Filipe Deschamps, que é um dos criadores de conteúdos sobre tecnologia e desenvolvimento para a web que mais admiro.

## Sobre o projeto
O Filipe Deschmaps há alguns anos criou um curso baseado em um grande projeto chamado [TabNews](https://www.tabnews.com.br/), que foi criado de forma aberta e colaborativa, tendo uma cacetada colaboradores, usuários e posts  sobre tecnologia, perrengues, a vida, o universo e tudo mais, como todo o bom dev sabe filosofar (tem realmente de tudo lá). Ou seja, é um projeto do mundo real.

E no curso, ele utiliza esse projeto do mundo real, que é realmente bastante complexo, como base para nos aprofundar em diversos conceitos de fundamentos de programação para a web e no processo, trabalhando a mentalidade da turma (nós, os alunos) para termos uma maturidade e uma mentalidade diferente na hora de abordarmos nossos projetos.

## Tecnologias utilizadas

Para o projeto vamos utilizar as seguintes ferramentas: 

- Node.js: Para rodar a aplicação;
- Git: Para o versionamento de código local;
- GitHub: Para hospedar nosso código de forma distribuída;
- Next.js: Como nosso framework para desenvolvimento web;
- React.js: Para criação das nossas interfaces;
- React Router DOM: Para podermos trazer vida para os nossos componentes na web;
